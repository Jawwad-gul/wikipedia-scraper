# Wikipedia Data Scraper

A Python web scraping project that extracts data from Wikipedia's "List of Largest Companies by Revenue" page and exports it to CSV format.

## 📋 Description

This project demonstrates web scraping capabilities using BeautifulSoup and requests to extract structured data from Wikipedia tables. The scraper pulls information about the world's largest companies including their revenue, employees, and headquarters.

## 🛠️ Technologies Used

- Python
- BeautifulSoup4 - HTML parsing
- Requests - HTTP requests
- Pandas - Data manipulation

## 📊 Data Extracted

- Company Rank
- Company Name
- Revenue (USD)
- Number of Employees
- Industry
- Headquarters Location

## 🚀 Usage

Run the Jupyter notebook `PullingDataFromWikiPedia.ipynb` to scrape the latest data from Wikipedia.

The scraped data will be exported to `exported_companies_data.csv`.

## 📝 Output

The project generates a CSV file containing all the scraped company data in a clean, structured format ready for analysis.
